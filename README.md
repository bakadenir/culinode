# CuliNode — Culinary Discovery App 🍽️

> **Flutter Mobile App:** Explore, discover, and save culinary spots with a beautiful Material Design interface.

---

## ✨ Features

- **🔍 Discover** — Browse culinary spots with rich visual cards
- **📱 Cross-Platform** — Android & iOS from a single codebase
- **🎨 Material Design 3** — Modern, clean UI
- **🗺️ Location-Aware** — Find nearby restaurants and food spots

---

## 🏗️ Tech Stack

| Layer | Technology |
|-------|------------|
| **Framework** | Flutter |
| **Language** | Dart |
| **Platform** | Android, iOS |
| **Design** | Material Design 3 |

---

